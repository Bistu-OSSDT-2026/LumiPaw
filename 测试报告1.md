## 4 路径依赖专项测试（本次Issue对应测试）
### 问题复现（修复前）
1. GitHub下载项目ZIP，解压至D盘非桌面文件夹
2. 执行python main.py，报错FileNotFoundError，找不到模块与图片素材
### 修复方案
删除硬编码桌面路径 `DESKTOP = Path.home() / "Desktop"`
新增动态项目根目录：`PROJECT_ROOT = Path(__file__).resolve().parent`
所有模块、素材路径统一使用 PROJECT_ROOT 拼接相对路径
### 验收测试结果
1. 全新电脑下载源码，仅执行 pip install -r requirements.txt + python main.py 即可启动
2. 不依赖本机固定桌面路径
3. 任务系统、经济模块、桌宠全部原有功能不受改动影响
4. 涉及修改文件：main.py
